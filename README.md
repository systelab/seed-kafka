## Getting Started

To get you started you can simply clone the `kafka-seed` repository:

### Prerequisites

You need [git][git] to clone the `kafka-seed` repository.

### Clone `kafka-seed`

Clone the `kakfa-seed` repository using git:

```bash

cd kafka-seed
```
